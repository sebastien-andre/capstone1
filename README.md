# Capstone project

## Requirements
- Python
- PlantUML

## Setup Instructions

### Clone the Repository
```bash
git clone https://github.com/sebastien-andre/capstone1.git
cd capstone1
```

### Install Dependencies
Using the `requirements.txt` file:
```bash
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```
